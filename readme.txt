Comandos disponibles en el MiniShell para CentOS:
------------------------------------------------

1. Comando: ls
   Funcionamiento: Muestra el contenido del directorio actual.

2. Comando: cd [ruta]
   Funcionamiento: Cambia el directorio actual al especificado por [ruta].

3. Comando: pwd
   Funcionamiento: Muestra la ruta completa del directorio actual.

4. Comando: mkdir [nombre]
   Funcionamiento: Crea un nuevo directorio con el nombre especificado por [nombre].

5. Comando: rm [ruta]
   Funcionamiento: Elimina un archivo o directorio en la ruta especificada por [ruta].

6. Comando: mv [ruta_origen] [ruta_destino]
   Funcionamiento: Mueve o renombra un archivo o directorio desde [ruta_origen] a [ruta_destino].

7. Comando: cp [ruta_origen] [ruta_destino]
   Funcionamiento: Copia un archivo o directorio desde [ruta_origen] a [ruta_destino].

8. Comando: ayuda
   Funcionamiento: Muestra la lista de comandos disponibles y su funcionamiento.

9. Comando: salir
   Funcionamiento: Termina la sesión del MiniShell y sale del programa.

Uso de los comandos:
-------------------

- Para ejecutar un comando, simplemente escribe el nombre del comando y, en algunos casos, los argumentos necesarios.

- Por ejemplo, para mostrar el contenido del directorio actual, utiliza el comando "ls".

- Para cambiar el directorio actual a una ruta específica, utiliza el comando "cd" seguido de la ruta deseada.

- Para crear un nuevo directorio, utiliza el comando "mkdir" seguido del nombre del directorio que deseas crear.

- Si necesitas más información sobre los comandos disponibles, utiliza el comando "ayuda".

- Para salir del MiniShell, utiliza el comando "salir".
